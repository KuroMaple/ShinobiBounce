## Project Description
Shinobi Bounce is a Naruto-inspired Pong-style game built in Unreal Engine using C++.

The goal is simple: take a timeless mechanic like Pong and transform it into something that evokes real nostalgia for Naruto fans — through atmosphere, UI, sound, and feel.

## Why?
I wanted to explore how far emotion and player experience can be pushed, even in the simplest games.

## Notes
This repository contains the **source code only** for the project.
Assets and full project files are not included.
